# PHP Library Template
A template repository for PHP libraries.

## Requirements
* Bash
* Git
* Docker

## Installation
```console
./install library-name
```

## Development Environment
Start up my development environment:
```console
laradockctl my:up
```

Shut down my development environment:
```console
laradockctl my:down
```

And more commands:
```
laradockctl --help
```
